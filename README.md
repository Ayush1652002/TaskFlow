# TaskFlow

A modern task management web app built with React — designed for productivity and simplicity.

🔗 **Live Demo:** [taskflow.vercel.app](#) ← update this after deploy

---

## Features

- ✅ Create, edit, delete tasks
- 📋 List view and Kanban board view
- 🎯 Task priority — Low, Medium, High
- 📁 Task categories — Work, Personal, College, Health
- 📝 Task descriptions
- 📅 Due dates with overdue highlighting
- 🔍 Search and filter tasks
- 📊 Analytics dashboard with charts
- 🌙 Dark SaaS UI
- 💾 LocalStorage persistence
- 📱 Mobile responsive
- ⚙️ Settings — name, default priority, clear tasks

---

## Tech Stack

- React (Vite)
- Tailwind CSS
- Context API
- React Router
- @dnd-kit (Drag & Drop)
- Recharts (Analytics)
- LocalStorage

---

## Getting Started
```bash
git clone https://github.com/yourusername/taskflow.git
cd taskflow
npm install
npm run dev
```
